## 内容摘要

- 面向对象思想，面向过程思想

面向对象是相对面向过程而言，面向对象和面向过程都是一种思想。

面向过程：强调的是功能行为。代表语言：C语言。

面向对象：将功能封装进对象，强调具备了功能的对象。代表语言：Java、C++、C#。

- 类
  - 属性：成员变量
  - 行为：成员函数
  - 构造函数
  - 成员变量和局部变量的区别
  - 对象内存结构
  - 匿名对象
  - 静态变量，静态函数
  - 静态代码块，构造代码块
  - 抽象类，接口
  - 内部类，匿名内部类


- 面向对象三大特征：封装、继承、多态
- this、super关键字
- 修饰符
  - public、private、protect
  - static、final、abstract
- 异常
